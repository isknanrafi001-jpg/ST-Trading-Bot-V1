# ST Trading AI Bot — GitHub Build Pack

No local Flutter installation is required.

## Build
1. Create a new GitHub repository.
2. Extract this ZIP.
3. Upload all extracted files and folders to the repository root.
4. Open Actions > Build ST Trading AI Bot > Run workflow.
5. After the green check, download `ST-Trading-AI-Bot-Production-Build`.

The artifact contains a universal APK, optimized APKs and a Play Store AAB.

## Current scope
This build contains the coded mobile foundation. Live login, market data, payment,
subscriptions and signal delivery require a deployed backend and authorized data source.
